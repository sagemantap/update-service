Stealth Miner Installer
=======================

📌 Fitur:
- Tanpa root
- Timer acak dan cooldown otomatis
- Anti-suspend
- Bersihkan jejak & cache
- Kamuflase trafik pertambangan

📦 Cara Menjalankan:
$ chmod +x autorun.sh
$ ./autorun.sh

📁 Struktur:
- stealth_miner.py      => Script utama
- autorun.sh            => Launcher otomatis
